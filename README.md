# Fire and Smoke Detection System

## Overview
This project is an IoT-based fire and smoke detection system using Arduino Uno.

The system detects smoke and fire using MQ2 and Flame sensors and sends SMS/call alerts using GSM SIM900A module.

## Features
- Smoke detection
- Flame detection
- SMS alerts
- Emergency phone call alerts
- Continuous monitoring
- Buzzer warning system

## Components Used
- Arduino Uno
- MQ2 Smoke Sensor
- Flame Sensor
- GSM SIM900A Module
- Buzzer
- Lithium-ion Battery

## Applications
- Home safety
- Industrial monitoring
- Laboratories
- Server rooms
